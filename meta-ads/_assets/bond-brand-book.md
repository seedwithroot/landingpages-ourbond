## Brand Foundation

### Brand Positioning

**Core Statement:**  
Elite-level personal security made accessible for everyone.

**Category Position:**  
Professional security company with consumer-friendly technology. Not just a panic button app — preventative 24/7 security agents in your pocket.

**Target Audiences:**
1. **Individuals** — Solo walkers, night-shift workers, students, travelers
2. **Families** — Parents tracking loved ones, household safety
3. **Lone Workers** — Delivery drivers, real estate agents, healthcare providers
4. **Students** — College/university safety, campus walking

### Brand Promise

When you activate Bond, you're never alone. Real security professionals — former FBI, Secret Service, NYPD — monitor, respond, and coordinate help before situations escalate.

### Brand Pillars

1. **Professional Expertise** — Real agents, elite-level training
2. **Accessible Technology** — One tap, no complexity
3. **Preventative Action** — Act before it reaches 911
4. **Human + AI** — Technology amplifies human judgment
5. **Surprisingly Affordable** — Elite security at consumer price point

### Color Usage by Context

#### Meta Ads (Social Media)

**Single Image Testimonials:**
- Background: Pure Black (`#000000`) or Off-Black (`#0A0A0A`)
- Text: Pure White (`#FFFFFF`) or Off-White (`#F5F5F5`)
- Accents: Gold (`#D4AF37`) for authority, Amber (`#FF9500`) for emphasis
- **Avoid:** Heavy use of branded colors (grey/blue/purple) — let testimonial be hero

**Feature Ads:**
- Primary: Bonus Blue (`#4169E1`) — emphasizes agent/technology relationship
- Background: Midnight Grey (`#2C2E3A`) or pure black
- Text: White

**Consumer-Facing Landing Pages:**
- Primary: Pacific Purple (`#8B7EC8`) — friendly, approachable
- Supporting: Bonus Blue for trust elements
- Backgrounds: White with purple accents

**Product/Technical Content:**
- Primary: Midnight Grey (`#2C2E3A`)
- Supporting: Bonus Blue for interactive elements
- Professional, sophisticated tone

### Color Ratio Guidelines

Based on strategic intent, use these approximate color ratios:

**Product Innovation Lab:** 80% Midnight Grey / 20% Bonus Blue / 0% Pacific Purple  
**Product Packaging:** 30% Grey / 50% Bonus Blue / 20% Purple  
**Bond Security Agent:** 20% Grey / 60% Bonus Blue / 10% Purple  
**Wordmark & Logo:** 18% Grey / 60% Bonus Blue / 35% Purple  
**Bond Member (Consumer):** 0% Grey / 20% Bonus Blue / 80% Pacific Purple

### Accessibility

**Contrast Requirements:**
- Text on background: Minimum 4.5:1 contrast ratio (WCAG AA)
- Large text (18pt+): Minimum 3:1 contrast ratio
- Interactive elements: 3:1 minimum

**Safe Combinations (Always Accessible):**
- White text on Midnight Grey ✅
- White text on Bonus Blue ✅
- White text on Deep Violet ✅
- Pure Black text on Off-White ✅
- Midnight Grey text on White ✅

**Avoid:**
- Light grey text on Bonus Blue ❌
- Pacific Purple text on white background ❌
- Amber on white (low contrast) ❌

## Typography

### Font System

**Primary Typeface:** Montserrat  
**Backup Typefaces:** Inter, Proxima Nova, DM Sans  
**Rationale:** Clean, modern, highly readable at all sizes. Professional without being corporate.

### Font Weights & Usage

| Weight | Name | Usage |
|--------|------|-------|
| 900 | Black | Hero headlines, major impact statements |
| 800 | ExtraBold | Primary headlines in ads |
| 700 | Bold | Subheadings, emphasized text |
| 600 | SemiBold | Secondary headlines, button text |
| 500 | Medium | Body text, quotes |
| 400 | Regular | Standard body text, captions |
| 300 | Light | De-emphasized text (use sparingly) |

### Type Scale (Meta Ads - 1080x1080)

**Desktop/Large Display:**
| Element | Size (pt) | Weight | Line Height | Letter Spacing |
|---------|-----------|--------|-------------|----------------|
| Hero Text | 110-140 | Black/ExtraBold | 1.0-1.1 | -3% to -4% |
| Primary Headline | 64-80 | Bold | 1.2 | -2% |
| Secondary Headline | 48-60 | SemiBold/Medium | 1.3 | 0% |
| Body/Quote | 40-54 | Medium/Regular | 1.4 | 0% |
| Attribution | 32-40 | Regular | 1.5 | 0% |
| Caption | 24-30 | Regular | 1.5 | 0% |

**Mobile Optimization:**
- Minimum readable size at thumbnail (300x300): 40pt for primary text
- Never go below 32pt for any text that carries key message

### Typography Best Practices

**Hierarchy:**
- Maximum 3 font sizes per composition
- Clear visual distinction between headline and body (2:1 size ratio minimum)
- Use weight variation before adding more sizes

**Alignment:**
- Left align: Multi-line quotes, longer text blocks
- Center align: Short statements, minimal designs, hero text
- Never right align or justify in ads

**Spacing:**
- Tight for impact: Headlines (-2% to -4% letter spacing)
- Normal for readability: Body copy (0% letter spacing)
- Generous line-height: Quotes (1.4x), body (1.5x)

**Pairing:**
- Montserrat + Montserrat: Use weight variation (Bold + Regular)
- Montserrat + Serif: Playfair Display or Lora for elegant contrast (use sparingly)

## Logo & Wordmark

### Logo Variations

**Primary Logo:**
- Full wordmark + icon
- Use when space allows
- Minimum size: 120px wide

**Icon Only:**
- When space is constrained
- Must be recognizable in context
- Minimum size: 80px × 80px

**Wordmark Only:**
- Horizontal lockup
- Text-heavy contexts
- Minimum width: 100px

### Logo Colors

**Standard Versions:**
1. **Multi-color** (18% Grey / 60% Blue / 35% Purple) — Use on white/light backgrounds
2. **White** — Use on dark backgrounds, photos with dark overlays
3. **Midnight Grey** — Use on light backgrounds when color is unavailable
4. **All Black** — Print, monochrome contexts

### Logo Placement (Meta Ads)

**Standard Position:**
- Bottom right corner
- 80-100px from right edge
- 80-100px from bottom edge

**Alternative Positions:**
- Bottom center (minimal designs)
- Top right (when bottom is occupied by visual)

**Size Guidelines:**
- Single Image Ads: 90-120px wide
- Carousel Ads: 80-100px wide per card
- Video Ads: 100-140px wide (first and last frame)

### Clear Space

Maintain clear space around logo equal to height of "B" in Bond wordmark on all sides. No text, graphics, or other elements in this zone.

### Logo Don'ts

❌ Don't stretch or distort proportions  
❌ Don't rotate logo  
❌ Don't apply drop shadows or effects  
❌ Don't place on busy backgrounds (add solid overlay first)  
❌ Don't alter colors or opacity below 90%  
❌ Don't place logo on colors that fail contrast requirements

## Photography & Visual Style

### Visual Direction

**Overall Aesthetic:**
- Cinematic, high-contrast
- Dark backgrounds (pure black or urban night)
- Dramatic but not threatening
- Professional photography feel, not stock

**Mood:**
- Confident, not fearful
- Empowered, not vulnerable
- Urban/real, not staged
- Warm undertones (golden hour, amber streetlight)

### Subject Direction

**People:**
- **Posture:** Walking with purpose, confident stride
- **Expression:** Aware but not anxious, composed
- **Avoid:** Looking over shoulder, frightened expressions, running
- **Clothing:** Relatable, urban casual (not luxury, not distressed)
- **Diversity:** Represent age, gender, ethnicity across ad sets

**Environments:**
- Urban settings: city streets, campus, parking areas
- Evening/night scenes preferred (safety context)
- Real locations, not studio setups
- Recognizable but not location-specific

**Lighting:**
- Warm keys (golden hour, amber streetlight)
- High contrast (dramatic shadows okay)
- Bokeh backgrounds (shallow depth of field)
- Face lighting: flattering, not harsh

### Photography Types by Ad Category

**Testimonials:**
- Portrait style (medium close-up to medium shot)
- Subject looking at camera or slightly off-camera
- Dark/black backgrounds
- Emphasis on face and expression

**Feature Demonstrations:**
- Subject with phone visible
- App interface glowing/visible on screen
- Environmental context (walking, in transit)
- Action-oriented but not blurred

**Lifestyle/Aspirational:**
- Subject in natural environment
- Phone present but not dominant
- Confident, empowered body language
- Warm, cinematic lighting

**Authority/Professional:**
- Formal portrait style
- Business attire or professional context
- Serious but approachable expression
- High-quality studio or environmental portrait

### Product Screenshots

**App Interface:**
- Always use current, production UI
- Keep screens clean (no fake notification clutter)
- Show real features, not placeholder content
- Ensure UI is readable even at small sizes

**Screen Treatment:**
- Slight glow/rim light effect okay
- Drop shadow for depth (subtle, 10-20% opacity)
- Avoid heavy Photoshop effects

### Image Sourcing

**Priority Order:**
1. Original photography/videography (ideal)
2. Licensed stock from premium sources (Unsplash, Adobe Stock)
3. User-generated content (with permission)

**Stock Photography Guidelines:**
- Avoid overly posed or obviously stock images
- Check that other brands aren't using same photo heavily
- Prefer candid, documentary-style shots
- Natural lighting beats studio setups

## Voice & Messaging

### Brand Voice

**Core Attributes:**
- **Confident** — Not fearful. We provide assurance, not alarm.
- **Accessible** — Professional security, consumer-friendly interface.
- **Direct** — Clear language, no jargon. Say what you mean.
- **Empowering** — Users feel safer and more in control, not dependent.

### Tone Variations by Context

**Top of Funnel (Awareness):**
- Relatable, understanding
- Problem-aware but not fear-inducing
- "You shouldn't have to walk alone."

**Mid-Funnel (Consideration):**
- Educational, informative
- Professional confidence
- "Here's how Bond works."

**Bottom Funnel (Conversion):**
- Direct, action-oriented
- Value-focused
- "Try free for 3 days."

**Remarketing:**
- Persistent, not pushy
- Address objections directly
- "Surprisingly affordable" (price), "Former FBI agents" (credibility)

### Messaging Frameworks

#### Problem → Solution → Outcome
"Walking home alone? [Problem] Bond monitors your route. [Solution] Never walk alone again. [Outcome]"

#### Authority → Insight → Invitation
"Former NYPD Commissioner Raymond Kelly [Authority] sees great value in preventative security services. [Insight] Try Bond free. [Invitation]"

#### Feature → Benefit → Why It Matters
"Track Me On The Go monitors your route. [Feature] Confirms safe arrival. [Benefit] Peace of mind every step. [Why It Matters]"

### Key Messages

**Differentiators (What Makes Bond Unique):**
1. Real agents, not just automation
2. Preventative, not just reactive (act before 911)
3. Elite expertise at consumer price point
4. Multiple protection layers (video, GPS, voice, bodyguards)
5. Surprisingly affordable

**Proof Points:**
- Advised by former Director of U.S. Secret Service
- Trusted by former NYPD Commissioner Raymond Kelly
- Former FBI, Secret Service, NYPD professionals
- 4.8★ rating, thousands of members
- 24/7 live agent availability

**Pricing Messages:**
- "Less than $1 a day"
- "Less than a gym membership"
- "Elite security for everyone"
- "Surprisingly affordable"
- "A bodyguard used to cost thousands. Now it's an app."

### Words to Use

✅ **Preferred Language:**
- Security agents (not "guards")
- Monitor, coordinate, respond
- Preventative, proactive
- Peace of mind, assurance
- Confident, safer, protected
- Elite, professional, trained
- Accessible, affordable

❌ **Avoid:**
- Panic, emergency (unless contextually appropriate)
- Danger, threat, attack (don't fear-monger)
- Guaranteed safety, 100% safe (can't promise absolutes)
- Spy, surveillance, tracking (negative connotations)
- Bodyguard (except in specific affordability context)

### Writing Style

**Sentence Structure:**
- Short sentences for impact
- Active voice ("Bond monitors your route" not "Your route is monitored")
- Present tense for immediacy

**Punctuation:**
- Em dash (—) for emphasis
- Period for authority
- Avoid exclamation points (feels desperate)

**Length:**
- Headlines: 3-7 words
- Pull quotes: 1-2 sentences max on image
- Ad primary text: 2-4 sentences
- Long-form: Break into scannable sections

## Ad Creative Guidelines

### Format-Specific Requirements

#### Single Image (1080x1080)

**Structure:**
- One clear message
- High contrast (text must be readable at thumbnail)
- 80-100px padding from all edges
- Logo present but subtle

**Text Limits:**
- On-image text: 1-2 sentences maximum (pull quote)
- Full message in primary text below image
- Headline: 40 characters or less

**File Specs:**
- Resolution: 1080×1080px
- Format: PNG (quality) or JPG (filesize)
- Color profile: sRGB
- File size: Under 30MB

#### Carousel (1080x1080 per card)

**Structure:**
- Card 1 is the billboard (strongest hook)
- Cards 2-5 build the case (each adds new value)
- Card 5 includes CTA
- Consider: Turn off auto card reordering if sequence matters

**Text per Card:**
- One key point per card
- 1-2 sentences maximum
- Progressive build (don't repeat same idea)

**Visual Consistency:**
- Same background treatment across cards
- Same font sizes across cards
- Visual throughline (color, layout) 

#### Video (Feed: 1:1, Stories: 9:16)

**Hook (First 5 Seconds):**
- First frame must stop the scroll
- Establish context immediately
- Create curiosity/emotion/recognition
- No slow intros, logo animations, or vague setup

**Body (6-25 seconds):**
- Fast-paced, keep moving
- Captions on (80% watch without sound)
- Show, don't just tell
- Build to CTA

**Duration:**
- Top-funnel cold: 15-30 seconds
- Mid-funnel education: 30-45 seconds
- Retargeting: 10-15 seconds

**File Specs:**
- Resolution: 1080×1080 (feed) or 1080×1920 (stories)
- Format: MP4 (H.264 codec)
- Frame rate: 30fps
- Max file size: 4GB (aim for under 100MB for fast loading)

### Call-to-Action (CTA)

**Preferred CTAs (In Order of Strength):**
1. "Try Free for 3 Days" — Creates urgency, low commitment
2. "Download Bond" — Simple, direct
3. "Get the App" — Similar to download, softer
4. "Start Your Free Trial" — Emphasizes no-risk

**Avoid:**
- "Learn More" (too vague)
- "Sign Up" (too transactional)
- "Get Started" (overused)

### Testing Strategy

**Test One Variable at a Time:**
- Video: Test hook type first
- Carousel: Test card 1, then card order
- Single image: Test one offer/visual/headline change

**Minimum Testing Threshold:**
- 1,000 impressions to same audience before evaluating
- CTR below 1% (warm audience) = pause and revise
- Wait 3-5 days for statistical significance
