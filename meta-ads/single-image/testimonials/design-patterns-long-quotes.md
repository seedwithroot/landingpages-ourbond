# Design Patterns for Long Testimonial Quotes

**Format:** 1080x1080 Single Image  
**Challenge:** Make long quotes readable and impactful on mobile

---

## Pattern 1: Pull Quote + Full Quote Split

**Concept:** Large pull quote on the image, full quote in primary text (ad copy below image)

### Visual Layout (1080x1080)

```
┌─────────────────────────────────┐
│                                 │
│  [Dark background/portrait]     │
│                                 │
│  " Ordering a bodyguard         │
│    was so easy on the app       │
│    — and surprisingly           │
│    affordable. "                │
│                                 │
│  — Ken P., Bond Member          │
│                                 │
│  [Small Bond logo]              │
└─────────────────────────────────┘
```

**Key Elements:**
- **On image:** ONLY the most powerful sentence (pull quote)
- **Ad primary text:** Full quote with context
- **Font size:** 60-80pt for pull quote, highly readable
- **Background:** Dark (black/charcoal) with portrait slightly visible, or pure black

**Example - Ken P. Affordability:**

**On Image (Large):**
```
"Ordering a bodyguard was so easy 
on the app — and surprisingly 
affordable."

— Ken P., Bond Member
```

**Primary Text (Under Image):**
```
"It never occurred to me to hire a bodyguard, but ordering one on Bond was so easy and surprisingly affordable that I gave it a try. It made everything less stressful and much more enjoyable." — Ken P., Lower Manhattan

Try Bond free for 3 days.
```

## Pattern 2: Visual Hierarchy (Size Contrast)

**Concept:** Emphasize the most important words in LARGE type, secondary context smaller

### Visual Layout

```
┌─────────────────────────────────┐
│                                 │
│  SURPRISINGLY                   │
│  AFFORDABLE                     │
│                                 │
│  "Ordering a bodyguard was so   │
│  easy on the app — and          │
│  surprisingly affordable."      │
│                                 │
│  — Ken P., Bond Member          │
│  [Bond logo]                    │
└─────────────────────────────────┘
```

**Key Elements:**
- Pull out 1-3 key words as HUGE headline (120-140pt)
- Rest of quote below in normal size (40-50pt)
- Creates visual anchor, makes scrolling thumb stop

**When to use:** When quote has a killer phrase that deserves emphasis

## Pattern 3: Split Screen - Portrait + Solid Quote Area

**Concept:** Half portrait, half solid color for clean text area

### Visual Layout

```
┌──────────────┬──────────────────┐
│              │                  │
│              │  "I see great    │
│   Portrait   │   value in       │
│   Photo      │   Bond's 24/7    │
│   (darker)   │   Personal       │
│              │   Security       │
│              │   Agents."       │
│              │                  │
│              │  — Raymond Kelly │
│              │  Former NYPD     │
│              │  Commissioner    │
└──────────────┴──────────────────┘
```

**Key Elements:**
- Left: Portrait (desaturated or darker overlay)
- Right: Solid color (black, deep navy, charcoal)
- Quote in white on solid area = maximum readability
- Name + title gets emphasis

**When to use:** Authority testimonials (Kelly, Craig), professional credibility

## Pattern 4: Oversized Quotation Marks

**Concept:** Giant quotation mark as design element, quote flows around it

### Visual Layout

```
┌─────────────────────────────────┐
│                                 │
│   "  I know Bond is with me     │
│      every step I take.         │
│                                 │
│      — Jennifer J.              │
│      Bond Member                │
│                                 │
│  [Background: her walking]      │
│  [Bond logo]                    │
└─────────────────────────────────┘
```

**Key Elements:**
- Opening quotation mark = 200-300pt (huge, semi-transparent)
- Quote itself = 50-60pt, readable
- Works best with SHORT quotes (Jennifer J.)
- Background image visible but subdued

**When to use:** Shorter, punchier quotes that deserve drama

## Pattern 5: Minimal White Space Design

**Concept:** Pure black background, lots of breathing room, quote is the ONLY thing

### Visual Layout

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│   "Bond gives me the            │
│   assurance I need to feel      │
│   safe anywhere I walk."        │
│                                 │
│   — Jennifer J., Bond Member    │
│                                 │
│                                 │
│                                 │
│   [Tiny Bond logo bottom]       │
└─────────────────────────────────┘
```

**Key Elements:**
- Pure black or white background (nothing else)
- Quote centered, generous padding all around
- Simple, clean, serif or clean sans-serif font
- Feels editorial, high-end

**When to use:** Powerful short quotes that need weight, not distraction

## Pattern 6: Text Highlight/Accent Color

**Concept:** Most of quote in white, KEY PHRASE in accent color (gold, amber, teal)

### Visual Layout

```
┌─────────────────────────────────┐
│                                 │
│  "As a former Chief of Police   │
│   with 30 years of experience   │
│   — I highly recommend Bond     │
│   to anyone seeking             │
│   GREATER SAFETY."              │
│                                 │
│  — Bill Craig                   │
│  Former Chief of Police         │
│                                 │
└─────────────────────────────────┘
```

**Key Elements:**
- Main quote: white text
- Key phrase: gold/amber (matches Bond brand)
- Dark background
- Creates focal point in longer quote

**Accent Colors:**
- Gold: #D4AF37 (premium, authority)
- Amber: #FF9500 (warm, safety)
- Teal: #00B9AE (tech, trust)

## Pattern 7: Two-Line Impact Statement

**Concept:** Force the quote into 2 huge lines max, even if you need to cut it

### Visual Layout

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│   SURPRISINGLY                  │
│   AFFORDABLE                    │
│                                 │
│   — Ken P., Bond Member         │
│                                 │
│   [portrait bottom third]       │
│   [Bond logo]                   │
└─────────────────────────────────┘
```

**Text on Ad (Primary):**
```
"Ordering a bodyguard was so easy on the app — and surprisingly affordable that I gave it a try." 

Ken P. ordered former FBI agents from the Bond app. Try free for 3 days.
```

**Key Elements:**
- Image carries 2-word punch
- Full story in ad copy below
- Works for mobile scrolling (stops thumb immediately)

## Recommended Approach for Your Bond Quotes

### Ken P. - "Surprisingly affordable"
**Use:** Pattern 2 (Visual Hierarchy) or Pattern 7 (Two-Line Impact)
- **On image:** "SURPRISINGLY AFFORDABLE" (huge)
- **On image subtitle:** "— Ken P., ordered former FBI agents from the app"
- **Primary text:** Full quote

### Raymond Kelly - Authority
**Use:** Pattern 3 (Split Screen Portrait)
- **Left:** Professional portrait (darker)
- **Right:** Pull quote on solid black
- **Emphasize:** "Former NYPD Commissioner" as credential

### Bill Craig - Recommendation
**Use:** Pattern 6 (Accent Color on Key Phrase)
- **Highlight:** "I HIGHLY RECOMMEND BOND" in gold
- **Rest:** White text on dark background

### Jennifer J. - Short & Sweet
**Use:** Pattern 4 (Oversized Quote Marks) or Pattern 5 (Minimal)
- Her quote is short enough to show IN FULL
- Let it breathe, don't over-design

## Typography Best Practices

**Font Pairings:**
- **Headline/Quote:** Montserrat Bold, Proxima Nova Bold, Inter Bold, DM Sans Bold
- **Attribution:** Same font, lighter weight OR contrasting serif (Playfair Display, Lora)

**Sizing for 1080x1080:**
- Hero text: 100-140pt
- Quote body: 50-70pt
- Attribution: 36-44pt
- Small Brand logo: 80-120px

**Line Height:**
- Quotes: 1.3-1.4x (tighter for impact)
- Attribution: 1.5x

**Letter Spacing:**
- Hero words: -2 to -4% (tighter, more impact)
- Body: 0% (default)

**Alignment:**
- Center: Works for minimal, short quotes
- Left: Better for longer quotes, feels less formal

## Production Checklist

Before finalizing any testimonial ad:

- [ ] Text readable at thumbnail size (preview at 300x300px)
- [ ] High contrast between text and background (4.5:1 minimum)
- [ ] Quote makes sense even if you only read the image (not the ad copy)
- [ ] Attribution clearly visible (name + title/role)
- [ ] Bond logo present but not competing with quote
- [ ] No more than 3 font sizes on the image
- [ ] Padding: minimum 80px from edges (text doesn't touch borders)
