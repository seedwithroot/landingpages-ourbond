# Testimonial Ad Variants - Extract & Bold Patterns

**Two Design Approaches for Long Testimonials**

---

## Design Pattern Overview

### Variant A: Extract + Full Review
- **On Image:** 2-4 impactful words (huge, top)
- **On Image:** Full review text below (readable size)
- **Ad Body:** Context or additional detail

### Variant B: Full Review + Bold Emphasis
- **On Image:** Full testimonial with KEY PHRASES in bold
- **Ad Body:** Call to action or additional context

## Production Guidelines

### Extract Words Pattern (Variant A)

**When to Use:**
- Short, punchy phrases exist in testimonial
- Want maximum impact at thumbnail size
- Clean, bold aesthetic needed

**Best Practices:**
- Extract 1-3 words maximum
- Words should convey core benefit
- Keep full quote below for context
- High size contrast (extract 2.5x larger than body)

**Color Treatment:**
- Extract words: Use accent colors (Gold, Amber, Teal) for emphasis
- Body text: White or off-white for readability
- Background: Pure black or dark for maximum contrast

---

### Bold Emphasis Pattern (Variant B)

**When to Use:**
- Quote flows well as complete statement
- Multiple key phrases throughout
- Want fuller story on image itself

**Best Practices:**
- Bold 3-5 key phrases (not everything)
- Bold phrases should be natural breaks in reading
- Don't bold single words in middle of sentence
- Maintain readability with consistent line height

**Color Treatment:**
- Bold text: Accent color (Gold for authority, Amber for warmth, Teal for tech) OR brighter white
- Regular text: Off-white for contrast with bold
- Option: Keep all white but use ExtraBold vs Medium weight difference

## Canva Build Instructions

### Variant A: Extract + Full Review

1. **Background:** Black (#000000)
2. **Add Extract Words:**
   - Text → Heading
   - Type your 1-3 extract words (e.g., "SURPRISINGLY AFFORDABLE")
   - Font: Montserrat Black, 120-140pt
   - Color: Choose accent (Gold #D4AF37, Amber #FF9500, or White)
   - Position: Top third of canvas, centered
3. **Add Full Review:**
   - Text → Subheading
   - Type full quote
   - Font: Montserrat Medium, 48-52pt
   - Color: Off-white (#EBEBEB)
   - Position: Below extract words
   - Line height: 1.4
4. **Add Attribution:**
   - Text → Subheading
   - Type name + title
   - Font: Montserrat Regular, 38-42pt
   - Color: Light grey (#AAAAAA)
5. **Add Logo:** Bottom right, 100px

---

### Variant B: Bold Emphasis

1. **Background:** Black or dark gradient
2. **Add Full Quote:**
   - Text → Heading
   - Type full quote
   - Font: Montserrat Medium, 48-54pt
   - Color: Off-white (#EBEBEB)
   - Position: Centered, generous padding
3. **Manually Bold Key Phrases:**
   - Select specific words/phrases in the text
   - Font: Change to Montserrat ExtraBold (keep same size)
   - Color: Gold (#D4AF37), Amber (#FF9500), or pure White (#FFFFFF)
   - Canva: Select text → Click character → Change font weight + color
4. **Add Attribution:** Same as Variant A
5. **Add Logo:** Bottom right or bottom center

**Pro Tip in Canva:**
- You can't directly bold within one text box easily
- Workaround: Create separate text boxes for bold phrases, position over regular text
- OR: Use different colored text for "bold" effect (easier in Canva)

## A/B Testing These Variants

**Test Setup:**
- Same audience, same budget
- Run Variant A vs Variant B for same testimonial
- 3-5 day test window
- Minimum 1,000 impressions per variant

**What You're Testing:**
- Does extract words (Variant A) stop scroll better?
- Does full review (Variant B) provide better context and convert?
- Which pattern resonates with your audience?

**Expected Results:**
- **Variant A:** Higher CTR (stops scroll), may have lower conversion if context missing
- **Variant B:** May have lower CTR, but higher quality clicks (more informed)

**Decision Criteria:**
- If cold audiences: Variant A likely wins (big words stop scroll)
- If warm audiences: Variant B may win (they need full story)
- If remarketing: Test both (depends on where they are in funnel)
