# Canva Quick Build Guide - Testimonial Ads

**Format:** 1080x1080px Instagram Post  
**Time:** 10-15 minutes per ad  
**Skill Level:** Beginner-friendly

---

## Setup in Canva

1. Go to Canva.com
2. Click "Create a design"
3. Select "Instagram Post" (automatically 1080x1080)
4. Start with a **blank canvas**

---

## BUILD 1: Ken P. - "Surprisingly Affordable"

### Step-by-Step

**1. Background**
- Click background
- Select solid color: Black (#000000)
- OR use gradient: Elements → Search "black gradient" → pick a subtle top-to-bottom gradient

**2. Add Hero Text**
- Click "Text" → "Add a heading"
- Type: **SURPRISINGLY**
- Press Enter
- Type: **AFFORDABLE**
- Select text → Font: **Montserrat Black** or **Bold**
- Size: **120**
- Color: White (#FFFFFF)
- Position: Upper-center of canvas
- Letter spacing: Click "Spacing" → -2 or -3

**3. Add Quote Text**
- Text → Add subheading
- Copy/paste this:
```
"Ordering a bodyguard was so easy 
on the app — and surprisingly 
affordable."
```
- Font: **Montserrat Medium** or **Regular**
- Size: **50-54**
- Color: Off-white (#EBEBEB)
- Position: Below hero text (leave some breathing room)
- Line height: Click "Spacing" → set to 1.3

**4. Add Attribution**
- Text → Add subheading
- Type: **— Ken P., Bond Member**
- Font: **Montserrat Regular**
- Size: **36-40**
- Color: Light gray (#B3B3B3)
- Position: Below quote, left-aligned or centered

**5. Add Bond Logo**
- Upload your Bond logo (white version)
- Resize to about 100-120px wide
- Position: Bottom right corner (leave ~60px margin from edge)
- Adjust transparency if needed (90% opacity looks premium)

**6. Final Adjustments**
- Click and drag elements to ensure 80-100px padding from all edges
- Check alignment: Select all text → Click "Position" → "Center" (if center-aligned design)
- Preview at small size: Zoom out to check readability

**7. Export**
- Click "Share" → "Download"
- File type: PNG (best quality) or JPG (smaller file)
- Download

## BUILD 2: Raymond Kelly - Split Screen Authority

### Step-by-Step

**1. Background Split**
- Background: Black (#000000)
- Click "Elements" → Search **"vertical split"** or **"two-tone background"**
- OR manually: Draw a rectangle covering right 50% of canvas, fill with black

**2. Add Portrait (Left Side)**
- Upload photo OR search "professional man suit" in Canva photos
- Position on LEFT half of canvas
- Resize to fill left 50%
- Click photo → "Edit image" → Adjust → Reduce saturation to -30 or -40 (make it less colorful, more serious)
- Add dark overlay: Click photo → "Effects" → "Shadows" → Adjust darkness

**3. Add Quote Text (Right Side)**
- Text → Add heading
- Copy/paste:
```
"I see great value in 
Bond's 24/7 Personal 
Security Agents — 
preventative services 
for situations that 
don't yet reach the 
severity of a 911 call."
```
- Font: **Montserrat SemiBold** or **Bold**
- Size: **48-56**
- Color: White (#FFFFFF)
- Position: RIGHT half only, left-aligned, generous padding from edge (~60px)
- Line height: 1.4

**4. Add Name (Gold)**
- Text → Add heading
- Type: **— Raymond Kelly**
- Font: **Montserrat Bold**
- Size: **42**
- Color: Gold (**#D4AF37**) ← Type this exact hex code
- Position: Below quote on right side

**5. Add Title**
- Text → Add subheading
- Type: **Former NYPD Commissioner**
- Font: **Montserrat Medium**
- Size: **34**
- Color: Light gray (#CCCCCC)
- Position: Below name

**6. Add Logo**
- Upload Bond logo → white version
- Position: Bottom right, ~80px from edges
- Size: ~100px

**7. Export**
- Download as PNG

## BUILD 3: Jennifer J. - Minimal Clean

### Step-by-Step

**1. Background**
- Click background → Solid black (#000000)
- No gradients, no images, pure simplicity

**2. Add Quote (Center Everything)**
- Text → Add heading
- Copy/paste:
```
"Bond gives me the 
assurance I need to 
feel safe anywhere 
I walk."
```
- Font: **Montserrat Medium** or **Regular**
- Size: **60-68**
- Color: White (#FFFFFF)
- Alignment: **Center**
- Position: Vertically centered on canvas (or slightly above center)
- Line height: 1.4
- Max width: Don't let text reach edges - keep it ~750-800px wide

**3. Add Attribution**
- Text → Add subheading
- Type:
```
— Jennifer J.
Bond Member
```
- Font: **Montserrat Regular**
- Size: **38-42**
- Color: Medium gray (#AAAAAA)
- Alignment: Center
- Position: Below quote, ~40-60px gap

**4. Add Logo**
- Upload Bond logo
- Position: Bottom center
- Size: ~90-100px
- Leave ~100px margin from bottom

**5. Check Spacing**
- Lots of empty space is the point
- Top padding: ~180-200px
- Bottom padding: ~180-200px
- This creates the "editorial" minimal feel

**6. Export**
- Download as PNG

## Font Settings Cheatsheet

| Element | Font | Size | Color | Spacing |
|---------|------|------|-------|----------|
| Hero Words | Montserrat Black/ExtraBold | 110-140 | #FFFFFF | -2 to -4 |
| Quote Body | Montserrat Medium/Regular | 48-60 | #FFFFFF or #EBEBEB | 1.3-1.4 line |
| Name | Montserrat Bold/SemiBold | 38-44 | #FFFFFF or #D4AF37 | 0 |
| Title/Attribution | Montserrat Regular | 32-38 | #AAAAAA or #CCCCCC | 0 |

## Quality Check Before Export

✅ **Readability Test**
- Zoom out to 25% in Canva
- Can you still read the main text clearly?
- If no → increase font size

✅ **Padding Check**
- No text touches the edge
- Minimum 80px space from all borders
- Looks balanced, not cramped

✅ **Contrast Check**
- Text stands out from background
- Easy to read even in bright sunlight
- High contrast = better mobile performance

✅ **Brand Consistency**
- Bond logo placed consistently across all ads
- Same fonts used (Montserrat family)
- Colors match brand palette

✅ **Phone Preview**
- Download image
- View on your actual phone
- If it looks good there, it's ready

## Time-Saving Tips

**Create a Brand Kit in Canva (Pro feature):**
- Upload Bond logo once
- Save color palette
- Save font combinations
- Reuse across all ads instantly

**Duplicate Designs:**
- Once you build Ken P. ad, click "File" → "Make a copy"
- Swap out text for Jennifer J. quote
- Keeps consistency, saves time

**Templates:**
- After building your first ad, save it as a template
- File → Publish as template
- Reuse structure, just change text/images
