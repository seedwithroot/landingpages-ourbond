## Production Recommendations

**Start with these 3:**
1. **Mockup 1** - Ken P. affordability (visual hierarchy) → Addresses price objection
2. **Mockup 2** - Raymond Kelly authority (split screen) → Builds credibility
3. **Mockup 3** - Jennifer J. minimal (clean/simple) → Use case, relatable

**Tools:**
- **Canva:** Use "Instagram Post" (1080x1080), custom design
- **Figma:** Start from blank 1080x1080 frame
- **Photoshop/Illustrator:** Standard social media template

**Fonts to Use:**
- **Primary choice:** Montserrat (Google Font, free, clean, professional)
- **Backup:** Inter, Proxima Nova, DM Sans, Poppins

**Export Settings:**
- Format: PNG or JPG
- Resolution: 72dpi (web)
- Color profile: sRGB
- Quality: 90-95% (balance file size and quality)

**Mobile Preview Test:**
- Before finalizing, view mockup at 375x375px (actual phone thumbnail size)
- If you can't read the hero text at that size, increase font size
